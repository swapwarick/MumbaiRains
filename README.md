# Mumbai Flood Digital Twin

A digital twin platform for simulating, analyzing, and visualizing urban flooding in Mumbai.

## Directory Structure

```text

│
├── data/

│   │
│   ├── osm/

│   │
│   ├── rainfall/
│   ├── drainage/
│   ├── rivers/
│   ├── tide/
│   └── output/
│
├── simulation/
│   ├── terrain.py
│   ├── rainfall.py
│   ├── runoff.py
│   ├── drainage.py
│   └── flood.py
│
├── backend/
├── frontend/
├── docs/
├── tests/
├── requirements.txt
└── README.md
```
